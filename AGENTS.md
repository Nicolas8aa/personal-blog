## Project Notes
- Package manager: `pnpm`.
- Common scripts: `pnpm dev`, `pnpm build`, `pnpm lint`, `pnpm format`.
- Site config lives in `src/config.ts` and `src/constants.ts`.
- Blog content lives in `src/data/blog`.

## Working Style
- Prefer small, focused diffs.
- Keep content edits in Markdown/MDX files.
- If you add new pages, update navigation in `src/config.ts` when needed.
